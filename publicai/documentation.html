API Documentation

Integration reference for the ZWIMA AI unified API.

Getting Started

  1. Create a free account — receive 500 starter credits.
  2. Create an API key in the API Key Center.
  3. Send your first request via Playground or POST /api/gateway/chat.
  4. Monitor usage in Usage and Credits.
  5. Upgrade plans in Billing when you need more volume.

REST API

The ZWIMA AI API is a REST interface for secure chat completions through OpenAI and Google Gemini.

Base URL: https://zwima-group.info

API Keys

Create keys in the API Key Center. Keys are shown once at creation — store them securely.

Pass your key in the request body as apiKey for gateway chat requests.

zw_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Authentication

Gateway chat requests pass your API key in the JSON body as apiKey. Dashboard APIs use a Bearer JWT from login.

Gateway: { "apiKey": "zw_live_..." }

Dashboard: Authorization: Bearer <session_token>

API terms: API Terms of Use

Endpoints

  • POST /api/gateway/chat — Chat completion gateway
  • GET /api/apikeys — API key management
  • GET /api/usage — Usage records

Chat Completion

Send a chat request with a provider-prefixed model identifier.

POST /api/gateway/chat

{
  "apiKey": "zw_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  "prompt": "Explain vector databases in simple terms.",
  "model": "gpt-4.1",
  "routingMode": "intelligent"
}

Example response:

{
  "content": "A vector database stores embeddings...",
  "provider": "OpenAI",
  "model": "GPT-4.1",
  "selectedModelId": "gpt-4.1",
  "routingReason": "Coding task routed to GPT-4.1.",
  "estimatedCost": 0.0042,
  "usage": { "inputTokens": 420, "outputTokens": 610, "totalTokens": 1030 },
  "creditsDeducted": 1030,
  "remainingCredits": 28720,
  "latencyMs": 612
}

Rate Limits

Default limits apply per API key and account tier. Early Access accounts receive 60 requests per minute and 1M tokens per day unless otherwise agreed.

Error Codes

  • 400 — Invalid request payload
  • 401 — Invalid or missing API key
  • 402 — Insufficient credits
  • 429 — Rate limit exceeded
  • 500 — Internal server error

Python SDK (example)

Install: pip install requests

import os, requests

API_KEY = os.environ["ZWIMA_API_KEY"]
resp = requests.post(
    "https://zwima-group.info/api/gateway/chat",
    json={"apiKey": API_KEY, "prompt": "Hello", "model": "gpt-4o"},
    timeout=60,
)
resp.raise_for_status()
print(resp.json()["content"])

Node.js SDK (example)

const API_KEY = process.env.ZWIMA_API_KEY;

export async function zwimaChat(prompt, model = "gpt-4o") {
  const res = await fetch("https://zwima-group.info/api/gateway/chat", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ apiKey: API_KEY, prompt, model }),
  });
  if (!res.ok) throw new Error(await res.text());
  return res.json();
}
